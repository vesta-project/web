<h3 align="center">Waitlist & Landing Page</h3>

<p align="center">
  This repository contains the source code for the <b>Vesta Launcher</b> landing page and early-access waitlist.
</p>

---

### About the Site

This is the front-facing web presence for Vesta. It handles:
- **Waitlist Signups** – Secure registration and referral tracking.
- **Referral System** – Priority-based access for users who refer friends.
- **Brand Experience** – Showcasing the aesthetic and core values of the launcher.

### Tech Stack

- **Framework** – [SolidStart](https://start.solidjs.com/) (Solid.js)
- **Styling** – CSS Modules with a custom design system.
- **Backend** – Supabase for database & Resend for email notifications.
- **Animations** – Motion (Solid) for smooth transitions.
